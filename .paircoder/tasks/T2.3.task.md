# Objective

Implement the Claude API few-shot transaction classification pipeline: 10-example assembly (user-confirmed + seed padding), confidence routing, user override persistence, and direct deposit detection that runs before classification to trigger the allocation engine.

# Implementation Plan

- `src/lib/classification/deposit-detection.ts`:
  - `isDirectDeposit(tx, user)`: positive amount + keyword match (`direct deposit`, `payroll`, `ACH credit`, `direct dep`) + amount ≥ `baseline_weekly_income * 0.60` + posted Friday (Sat/Sun fallback)
  - Called first in pipeline; on match sets `is_direct_deposit = true` and triggers allocation engine

- `src/lib/classification/prompt.ts`:
  - `assembleExamples(userId)`: fetch up to 10 most recent user-confirmed classifications; pad from `seed_examples` JSONB if < 10
  - `buildClassificationPrompt(tx, buckets, examples)`: returns the full prompt string
  - Seed examples from spec: DoorDash/delivery (amount heuristic > $30 → Food, ≤ $30 → Flex), gas stations → Flex, grocery stores → Food, utility keywords → Bills, minimum payment keywords → Debt

- `src/lib/classification/pipeline.ts`:
  - `classifyTransaction(txId)`: main entry; runs deposit detection first; calls Claude API; routes by confidence
  - Claude API call: `@anthropic-ai/sdk`, model `claude-sonnet-4-6`, JSON output `{ bucket_id, confidence, rationale }`
  - Confidence routing: ≥ 0.85 → auto-assign; 0.60–0.84 → assign + store confidence; < 0.60 → null

- `src/app/api/transactions/classify/route.ts`: POST handler called by webhook after transaction stored
- `src/app/api/transactions/[txId]/override/route.ts`: sets `classification_override = true`; updates `bucket_id`

# Acceptance Criteria

# Verification

- Cold-start: first transaction → all 10 examples come from `seed_examples`
- After 10 user-confirmed transactions: all 10 examples come from user history
- DoorDash $25 → Flex bucket (≤ $30 heuristic)
- DoorDash $45 → Food bucket (> $30 heuristic)
- Direct deposit Friday → `is_direct_deposit = true`, allocation engine triggered
